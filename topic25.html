📘 Екіөлшемді массивтер (матрица)

Екіөлшемді массивтер (матрица) – жолдар мен бағандардан тұратын, элементтері екі индекс арқылы анықталатын деректер құрылымы. Яғни бірөлшемді массив бір ғана тізім болса, екіөлшемді массив кесте (таблица) түрінде беріледі. Мұндай массивтің әрбір элементі екі координата арқылы беріледі: бірінші индекс – жол нөмірі, екінші индекс – баған нөмірі. Мысалы, A[2][3] дегеніміз 2-жол, 3-бағандағы элементті білдіреді. Матрицалар көбінесе математикалық есептерде, кестелік мәліметтерді сақтау мен өңдеуде қолданылады. Мысалы, оқушылардың бағалары, температура кестесі немесе шахмат тақтасы – бәрі екіөлшемді массивке мысал бола алады. Екіөлшемді массивті жазу мысалы: ```id="mat1" int A[3][3] = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; ``` Бұл жерде 3 жол және 3 баған бар матрица берілген. Элементтерге қол жеткізу үшін екі индекс қолданылады, мысалы A[1][2] = 6 (2-жол, 3-баған). Екіөлшемді массивтермен жұмыс істеу үшін көбінесе екі цикл (ішкі және сыртқы) қолданылады: ```id="mat2" for(int i = 0; i < 3; i++) { for(int j = 0; j < 3; j++) { // элементпен жұмыс } } ``` Мұнда сыртқы цикл жолдар бойынша, ал ішкі цикл бағандар бойынша жүреді. Екіөлшемді массивтің негізгі қасиеті – мәліметтерді кесте түрінде сақтап, оларды ыңғайлы өңдеуге мүмкіндік беру. Қорытындылай келе, екіөлшемді массив – жолдар мен бағандардан тұратын құрылым, ол күрделі мәліметтерді ұйымдастыруда өте маңызды рөл атқарады.
