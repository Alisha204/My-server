📘 Кірістірілген шартты операторлар

Кірістірілген шартты операторлар — бұл бір шартты оператордың ішінде тағы бір шартты оператордың орналасуы. Яғни, бір шарт тексерілгеннен кейін, оның нәтижесіне байланысты ішкі қосымша шарт тексеріледі. Бұл тәсіл бірнеше деңгейлі тексерулер жасау қажет болған жағдайда қолданылады. Алгоритмнің жалпы идеясы: алдымен негізгі шарт тексеріледі. Егер ол ақиқат (true) болса, оның ішінде орналасқан екінші шарт тексеріледі. Осылайша, шарттар бірінің ішінде бірі орналасып, күрделі логикалық шешімдер қабылдауға мүмкіндік береді. Мысалы, оқушының бағасын анықтау керек делік. Егер балл 50-ден жоғары болса, онда ол өтті деп есептеледі. Бірақ нақты бағаны анықтау үшін қосымша шарттар қолданылады. Псевдокод түрінде: Басы балл енгізу егер балл ≥ 50 болса  егер балл ≥ 90 болса → "5"  әйтпесе егер балл ≥ 75 болса → "4"  әйтпесе → "3" әйтпесе  "2" Соңы JavaScript тілінде мысал: ```javascript id="k83m2d" let score = 82; if(score >= 50){ if(score >= 90){ console.log("Баға: 5"); } else if(score >= 75){ console.log("Баға: 4"); } else { console.log("Баға: 3"); } } else { console.log("Баға: 2"); } ``` Бұл жерде: * бірінші `if` — негізгі шарт * ішіндегі `if` — кірістірілген шарт Қорытындылай келе, кірістірілген шартты операторлар күрделі есептерді шешуге көмектеседі және бірнеше шартты бір-біріне тәуелді түрде тексеруге мүмкіндік береді.
