📘 Массив элементтерін сұрыптау алгоритмін жазу

Массив элементтерін сұрыптау алгоритмі дегеніміз массивтегі элементтерді белгілі бір тәртіппен, яғни өсу немесе кему ретімен орналастыру процесі болып табылады, бұл процесс программалауда өте маңызды орын алады, себебі деректерді ретке келтіру іздеуді, талдауды және өңдеуді жеңілдетеді, сұрыптау алгоритмдері әртүрлі болады және олардың әрқайсысының өз ерекшелігі, жылдамдығы және қолдану аймағы бар, ең қарапайым сұрыптау әдістерінің бірі көпіршік (bubble sort) сұрыптауы болып табылады, бұл әдісте массивтің көршілес элементтері бір-бірімен салыстырылып, егер олардың орналасу реті дұрыс болмаса орындары ауыстырылады, бұл процесс бірнеше рет қайталанып, ең үлкен элемент біртіндеп массивтің соңына жылжиды, кейін келесі элементтер дәл осылай реттеледі, нәтижесінде массив толық сұрыпталады, бұл алгоритмнің жұмыс істеуі үшін әдетте екі цикл қолданылады, сыртқы цикл барлық элементтер бойынша өтеді, ал ішкі цикл көршілес элементтерді салыстырады, мысалы int arr[5] = {5, 2, 9, 1, 3}; for(int i = 0; i < 5; i++) { for(int j = 0; j < 5 - 1; j++) { if(arr[j] > arr[j + 1]) { int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } бұл жерде әрбір айналымда массивтің элементтері салыстырылып, орындары ауыстырылып отырады, нәтижесінде бастапқы 5 2 9 1 3 массиві 1 2 3 5 9 түрінде сұрыпталады, сұрыптау алгоритмдерінің негізгі мақсаты деректерді белгілі бір ретке келтіру болып табылады және олар көптеген есептерде қолданылады мысалы деректер базасында, іздеу жүйелерінде және әртүрлі қолданбаларда, көпіршік сұрыптауы қарапайым болғанымен оның кемшілігі бар себебі ол баяу жұмыс істейді әсіресе элементтер саны көп болған кезде себебі оның уақыттық күрделілігі шамамен n²-қа тең, сондықтан үлкен деректер үшін тиімді емес, бірақ үйрену үшін ең ыңғайлы алгоритмдердің бірі болып табылады, қорытындылай келе массив элементтерін сұрыптау алгоритмі деректерді ретке келтірудің маңызды тәсілі болып табылады және көпіршік әдісі солардың ең қарапайым әрі түсінікті түрі болып саналады.
